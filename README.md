# People Database with Status Toggle

Simple PHP & MySQL project to store people’s names and ages, with the ability to toggle their status (0 or 1) in real time using JavaScript (Fetch API).

## Features
- Add name and age via a form
- Store in MySQL table people
- Display records in a table
- Toggle status without page reload

## Technologies
HTML – CSS – PHP – MySQL – JavaScript – XAMPP

## How to Run
1. Start XAMPP (Apache & MySQL)
2. Create DB mydatabase and table people
3. Copy files to htdocs/myproject
4. Go to http://localhost/myproject/
